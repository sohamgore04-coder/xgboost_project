# XGBoost Prediction App

This project is a simple **Machine Learning Web Application** built using **XGBoost** and **Streamlit**.

The app takes input values from the user and predicts an output using a trained machine learning model.

The goal of this project is to demonstrate how a machine learning model can be trained and then deployed as a simple web application.

# Project Overview

In many real-world problems, we want to predict a value based on some input information.

For example:
- Predicting house prices
- Predicting sales
- Predicting health indicators
- Predicting fitness progress

This project shows the complete process:

1. Train a Machine Learning Model  
2. Save the trained model  
3. Build a simple web interface  
4. Allow users to make predictions  

---

# Technologies Used

This project uses the following tools:
